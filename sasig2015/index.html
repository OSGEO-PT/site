O nosso SASIG ♥ está de volta

Em 2015 acontece a 6ª edição da SASIG - conferência nacional de software aberto para Sistemas de informação geográfica - organizada pelo Capítulo Português da OSGeo - Open Source Geospatial Foundation.

O SASIG é essencialmente um espaço de partilha de experiências onde todos - mais ou menos experientes - os que querem conhecer e aprender mais sobre software aberto para Sistemas de Informação Geográfica são bem vindos.

O SASIG deste ano é em Lisboa, mas onde?

Muito em breve vamos revelar a localização do evento, do local em concreto, com coordenadas e tudo, por isso fiquem atentos!!!

Programa do SASIG

Fica atento, estamos a "cozinhar" um SASIG especial para ti!

Temas a abordar no SASIG:

- Novos desenvolvimentos em software aberto para SIG;
- Transição de software SIG proprietário para aberto;
- Interoperabilidade e standards;
- Formação e suporte em software SIG aberto;
- Modelos de negócio para software SIG aberto;
- Dados, conteúdos e arquiteturas abertas;
- Casos de estudo de software aberto aplicados às mais diversas áreas;
- Informação geográfica voluntária na produção de cartografia;
- Infraestruturas de dados espaciais suportadas por software aberto;
- Outros temas multidisciplinares transversais à temática do software aberto para SIG.

Patrocionadores do sasig

Seja um dos nosso patrocinadores, acesse aqui ao guia do patrocinador!

Procuramos patrocinadores para um evento único no sector SIG e Open Source em Portugal.
